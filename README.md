# Currículo Web - Projeto Faculdade - Engenharia de Software
Este é um projeto de currículo web desenvolvido como parte do curso de Engenharia de Software. Ele utiliza HTML, CSS e JavaScript para criar uma página de currículo interativa e responsiva.

## Recursos
- Apresenta informações pessoais, qualificações e experiência profissional.
- Layout responsivo que se adapta a diferentes tamanhos de tela.
- Navegação suave usando âncoras internas.
